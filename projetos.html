Projetos

Veja abaixo alguns dos meus projetos:

Projeto de Gráficos em C (ASCII)

Este projeto se propõe a criar gráficos em C no terminal com uma resolução fixa. Ele serviu como maneira de estudar o uso de argumentos.

YT-DLP-GUI

Este projeto, feito em python, serve para baixar vídeo e áudio do youtube utilizando Yt-DLP e FFMPEG. O objetivo dele foi estudar Tkinter e o uso de bibliotecas externas utilziando Python.

Calculadora de pesos do SISU

Este projeto calcula a média ponderada do SISU a partir das notas do ENEM utilizando os pesos da Universidade Federal de Sergipe (UFS) - Campus Sede (José Aloísio de Campos) em 2024. Os dados foram obtidos a partir do termo de adesão divulgado pela universidade e processados com Power Automate e Microsoft Excel.

Feito por Gustavo Gomes.
